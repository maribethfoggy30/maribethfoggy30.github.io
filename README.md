# Sam Stryker Portfolio Website

A responsive personal portfolio and research website built for GitHub Pages.

## Files

- `index.html` — all website content
- `styles.css` — layout, typography, colors, and mobile styling
- `script.js` — mobile navigation, scroll animations, and automatic footer year
- `assets/favicon.svg` — browser tab icon
- `assets/Sam-Stryker-Resume.pdf` — replace this placeholder with your actual résumé

## Before publishing

Search `index.html` and replace:

- `YOUR_LINKEDIN_URL`
- `YOUR_EMAIL_ADDRESS`

Then add your résumé at:

`assets/Sam-Stryker-Resume.pdf`

The résumé link will not work until that file is added.

## Upload to GitHub

In the `samstryker.github.io` repository:

1. Delete or replace the current `index.html`.
2. Upload `index.html`, `styles.css`, `script.js`, and the `assets` folder.
3. Commit the changes.
4. Wait about one minute and refresh `https://samstryker.github.io`.

## Editing content

Most wording is in `index.html`. You can edit it directly on GitHub with the pencil icon.
